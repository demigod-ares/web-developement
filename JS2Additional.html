Interaction (alert, prompt, confirm)

Alert function in Javascript pops up a message string in your browser. The pop up is shown in a modal window, you can't access the page content unless you close modal window. Alert function doesn't return anything. On the contrary, prompt displays a message and asks for the user input. It is optional to provide default user input along with the message in prompt. Prompt returns same value as user input. In addition, confirm asks a Yes or No question and returns true or false respectively.

Timing methods!

The setTimeout() method calls a function or evaluates an expression after a specified number of milliseconds. This function is only executed once. Use the clearTimeout(timeOutID) method to prevent the function running. setInterval() method calls a function or evaluates an expression at specified intervals in milliseconds. It will continue calling the function until clearInterval() method is called or the window is closed.

JSON in JavaScript

Now, we are going to learn about the concept of JSON in JavaScript. JSON is a format for storing and transporting data. JSON is often used when data is sent from a server to a web page.So, what is JSON?

  • JSON stands for JavaScript Object Notation
  • JSON is a lightweight data-interchange format
  • JSON is language independent
  • JSON is "self-describing" and easy to understand
The JSON syntax is derived from JavaScript object notation syntax, but the JSON format is text only. The code for reading and generating JSON data can be written in any programming language.
