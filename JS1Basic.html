Variables, Data Types & Operator in JavaScript

Check the script in the source code. It will show that there are no identifiers or keywords to define data types. This is called as dynamic typing, where JavaScript will automatically assign data types. Note that variable c is not declared with keyword "var", but still it works. In addition, redeclaring any variable is not allowed but reassignment is allowed.

There are 2 types of operator in JS. They are Unary operator and Binary operator.

  • Unary operator work on single operand
  • Binary operator operates on single operand.
Note that concatination is allowed inside console.log() function using + operator.

Scope and Functions

Unlike other languages, redeclaring a variable in JS doesn't gives an error. We use 'let' keyword to keep the variable limited to the block. Try to use 'let' instead of 'var'. However, you may use 'const' keyword to declare fixed values and changing these values will give an error. Functions are used for code reusablity. Functions also have the same implementation in JS as that of C language.

Strings and String functions in JavaScript

It's easy to deal with Strigs in Java Script. It is interesting to understand the use of `` quotes. Escape sequence like \n is also available in JavaScript. There are methods to dynammically add to the content of html element using their class and ID using strings. But we will focus on more important "string functions".

  • Number of character in a String is given by String.length function.
  • Index of first occorance of subset of string is given by String.indexOf("item to be searched") function.
  • A subset of string is extracted using String.slice(starting position, ending position) function. It can take negative values as well.
  • String.substring(starting position, ending position) function is similar to above, but it allows positive values only.
  • A similar application is String.substr(starting position, number of characters) function.
  • Contents of a string can be changed using String.replace("Replaced string","Replacement string") function.
  • String can be transformed to uppercase or lowercase using String.toUpperCase() or String.toLowerCase() function respectively.
  • String.concat("String to add") is useless function because we have + operator.
  • String.trim() function is used to remove white spaces from beginning and the end in a string.
  • String.charAt(index) function is used to get the character at given index of string.

Conditionals(if-else & switch-case)

if-else and switch-case statements in JS works same as C. It is to be noted that switch-case is prefered over if-else because of smaller time complexity. break statement also have the same task as that in c language.

Objects contains lot of information in terms of key value pairs. A similar implementation is dictionary in python. Array are declared in a similar fashion as in C. There are methods to access individual element of an Array or an object. Their are various functions for Array. One important function is String.push("item to be pushed") which changes the String.
